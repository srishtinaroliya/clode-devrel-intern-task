# FAQ / Objections – Clode

1. Is Clode another internal developer platform?  
   No. Clode focuses on automating operations, not just providing portals.

2. How is Clode different from Humanitec?  
   Clode executes automation instead of orchestrating configurations.

3. Do I need Kubernetes to use Clode?  
   No. Clode supports multiple cloud setups.

4. Is Clode AI-first?  
   Yes. Automation decisions are AI-driven.

5. Can Clode replace DevOps engineers?  
   No. It removes repetitive manual work.

6. Is this for startups or enterprises?  
   Both — especially growing engineering teams.

7. How does Clode compare to consulting firms?  
   Software scales automation better than services.

8. Do I need to manage YAML files?  
   Minimal configuration is required.

9. Does Clode handle monitoring?  
   Yes, as part of operations automation.

10. Is Clode opinionated?  
    Yes — towards automation over manual ops.
