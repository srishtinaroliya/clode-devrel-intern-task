# DevRel / Developer Advocate Intern Task – Clode

This repository contains my submission for the DevRel / Developer Advocate Intern task at Clode.

The goal of this task is to analyze competitors in the cloud, DevOps, and platform engineering space, and translate those insights into DevRel-ready messaging for developers.

---

## About Clode

Clode is an AI-powered cloud automation platform that helps teams:

- Deploy applications
- Manage cloud infrastructure
- Automate DevOps workflows
- Reduce manual operational effort

Clode focuses on automating the **later stages of the SDLC**:
**build → deploy → operate → monitor**

- DevRel content (Tweets, LinkedIn posts, Blog outline)
